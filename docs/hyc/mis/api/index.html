    会养车后台 API 介绍

    以下为会养车后台 所有接口说明。

    共性：

    • 统一采用 JSON 数据通信。
    • 统一采用 POST 方式请求。

    • 固定的数据返回格式：

      • error_code 0 或者其他数字，当值为 0 时表示成功！
      • error_message 信息说明，用来描述错误原因。
      • data 数据体。

    • 固定的列表请求数据返回格式。

      • error_code 0 或者其他数字，当值为 0 时表示成功！
      • error_message 信息说明，用来描述错误原因。
      • data
        • pageInfo
          • alarmCount 数据数目总数
          • currentPage 当前页码
          • totalPage 总页码
      • list
        • 数据体 1
        • 数据体 2

    响应数据属性说明

    基本信息

    • providercode 商户编码
    • providerName 商户名称
    • providerAddress 商户地址
    • postcode 邮编
    • area {Number} 地区 id
    • areaObject {Object}
      • city 城市如：“北京”
      • district 城区如： “丰台区”
      • area 地区 “方庄”
    • personname 联系人
    • cellphone 手机号码
    • telephone 预约电话
    • wechat 微信号
    • qq QQ号
    • website 商户网址
    • commercialtype 经营类型: （单店直营、连锁直营、加盟品牌、其他）单选
    • chaintype 商户性质: （洗车店、美容装饰店、汽车养护店、修理厂、4S店、其他）单选
    • officetype 物业类型:（住宅区、写字楼、加油站内、商业中心、其他）单选
    • qualification 具备资质:（营业执照、税务登记证、组织机构代码、修理行业资质）多选 用分号并列分隔
    • longitude {Float} 商户GPS信息（经度)
    • latitude {Float} 商户GPS信息（纬度）
    • keyword 补充关键词（文字，多个词组可用分号分隔，用于补充店铺特色便于准确搜索）
    • collaboration 现有合作平台（手工填写文字）
    • description 商户介绍
    • logo 商户Logo（图片地址）一张
    • logoDesc Logo描述
    • picture 商户图片地址（多张图片最多5张）
    • picDesc 商户图片描述

    结算信息

    • settlementday 结算日（从10-20日选择一天）
    • iscompany 对公对私
    • bankaccountname 公司名称 | 个人名称
    • bankaccountnumber 公司账号 | 个人银行账号（数字）
    • bankname 公司_银行行号（数字）
    • bankbranchname 公司开户行名称| 个人开户行名称

    店面信息

    • opentime 营业时间（格式HH:MM-HH:MM）
    • shoptype 店面类型:（单层门面、两层门面、多层门面）单选
    • servicedescription 服务类型（ 洗车、美容、装饰、改装、保养、快修、商品销售、车险服务、休息区、道路救援）多选 字符串用分号分隔开，上传，例如：洗车；美容；
    • employeenumber 员工数目（数字）
    • businesscondition 营业情况（日均洗车（留空格，填数字）辆、日均车流量（留空格，填数字）辆、月均营业额（留空格，填数字）元）字符串用分号分隔开，上传
    • equipment 门店设备（洗车机、扒胎机、举升机、四轮定位设备、烤漆房、无尘贴膜车间）多选 字符串用分号分隔开，上传，例如：洗车；美容；
    • creditrank 综合评分（0-5）含0.5分，十个档位
